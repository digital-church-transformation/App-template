# App-template

## Preparation Steps

1. Install https://nodejs.org/en/
2. Install Ionic: 
```
npm install -g @ionic/cli
```
3. Clone Repository: 
```
git clone https://github.com/digital-church-transformation/App-template.git
```
4. Serve Web Version: 
```
cd App-template
ionic serve
```
